== Description ==

= How Easy Mailchimp Forms helps your website =

Set up and start to use in minutes. Insert your MailChimp Forms easily to your site. Use short codes to insert forms to your posts or pages. Use our custom widget to insert forms to your widget areas. Select your list, pick a style and you are ready.

You can read more about the plugin here: [Easy MailChimp Forms For WP](http://marketingv8.wordpress.com/)

= Plugin Highlights = 

*   Just one "technical" thing need: the MailChimp API key. You can read more about it here: [MailChimp API key](http://kb.mailchimp.com/article/where-can-i-find-my-api-key/)
*   You can use your subscription forms in any widget area after a few clicks 
*   You can insert your MailChimp forms to your posts or pages with the help of a shortcode wizard
*   The plugin includes styles for the forms! You can set the MailChimp forms styles with a single click
*   The included styles are fully responsive

    You should set the MailChimp form's inputs in your MailChimp account. Then go to your WordPress site and insert the forms easily with this plugin to your pages, posts or widget areas.

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.

1. You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory the top-level folder. Don't just upload all the php files and put them in `/wp-content/plugins/`.
1. Activate the plugin through the 'Plugins' menu in WordPress wp-admin
1. Visit your Easy Mailchimp Forms options (*Settings - Easy MailChimp Forms for WP*) - required to insert your MailChimp API key
1. Done!

That's all. You can use the plugin.
